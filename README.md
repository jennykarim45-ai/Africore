# Africore

Mission Data effectué dans le cadre de la formation chez Wild Code School. 

# Présentation

Les Missions DATA sont une nouvelle méthodologie d’apprentissage, ciblée sur les besoins métiers, qui a été testée et validée par des promos Data Analyst. 
Cette méthodologie concentre jeu de rôle, analyse de données et relation client. 
C’est une méthodologie complémentaire aux DOJOS

# Les étapes préliminaires

Chaque élève choisit un dataset en fonction de son centre d’intérêt (sur Kaggle, Data.gouv.fr ou autre )
Le dataset doit être suffisamment complexe pour permettre une étude complète, mais ne pas nécessiter de nettoyage trop complexe vu le temps dédié .
Une fois le dataset validé par le formateur, chaque élève doit rédiger un Brief Client. Celui-ci doit comporter, en plus du lien vers le dataset : 
- Le nom et la description du client fictif
_ La nature de la mission data
